Indicator Docs: Adaptive Dual MA Trend Ribbon

Overview

The Adaptive Dual MA Trend Ribbon is a two-line moving-average ribbon designed to highlight bullish / bearish trend phases and neutral zones using configurable SMMA / EMA / SMA / WMA pairs plus an optional ATR-based filter. It is inspired by community “Larsson-style” trend ribbons but remains fully open-source and transparent.

Default Settings (recommended starting point)

SMMA “Larsson-style” preset

  • Source: HL2
  • MA Type: SMMA (RMA)
  • Fast Length: 15
  • Slow Length: 29
  • Use ATR Neutral Filter: ON
  • ATR Length: 60
  • ATR Multiplier: 0.3 – 0.5

EMA “React quicker” preset

  • Source: HL2 or Close
  • MA Type: EMA
  • Fast Length: 30
  • Slow Length: 60
  • Use ATR Neutral Filter: ON
  • ATR Length: 60
  • ATR Multiplier: 0.3

How to use it

  1. Apply the indicator and start with one of the presets.
  2. Treat orange as bullish, blue as bearish, gray as neutral.
  3. Use the ribbon as a context filter, not a signal generator.
  4. Longer EMAs for higher timeframes (50/100), shorter EMAs for intraday (10/21).

Things to keep in mind

  • All moving averages lag.
  • Neutral zones help filter chop but are not perfect.
  • Always combine with risk management.
