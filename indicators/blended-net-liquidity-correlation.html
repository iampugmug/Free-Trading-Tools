Blended Net Liquidity Correlation

Many traders use liquidity-based models to understand broad macro conditions and their potential influence on price. This indicator provides a transparent, configurable net-liquidity framework built from public FRED series. The indicator is free to use; the script is protected.

How Traders Recreate This Without the Indicator

Traders often approximate similar liquidity views manually using TradingView symbol math and public data series. For example, you can combine multiple series using + / − / × / ÷ to build your own “net liquidity” proxy and overlay it on price.

  1. Use TradingView symbol math to combine public series into one line.
  2. Apply smoothing to reduce noise.
  3. Compare against price and test different offsets/timeframes.

This indicator packages that workflow into a cleaner structure with component toggles, blending controls, smoothing options, and correlation windows.

Overview

Many traders use liquidity-based models to understand broad macro conditions and their potential influence on price. While institutional-grade models often rely on proprietary inputs, the community frequently studies public FRED series—such as WALCL, RRP, TGA, and BTFP—to construct transparent net-liquidity frameworks that approximate capital movement within the financial system.

This script does not replicate or reverse-engineer any commercial macro indicator. Instead, it provides a fully open research framework for exploring how different liquidity components interact — and how those liquidity dynamics correlate with market price across various time windows. All data comes from standard FRED series, and all calculations are fully disclosed for transparency at the behavior/input level.

What the Script Does

1. Builds a customizable blended net-liquidity model

  • Includes WALCL, RRP, TGA, BTFP, ΔTOTLL, and ΔM2
  • Allows turning each component on/off
  • Provides adjustable weighting for WALCL vs non-WALCL models
  • Smooths liquidity using optional SMA smoothing
  • Supports time-offsetting liquidity to study lagging effects

2. Runs multi-period price correlations

  • 30D, 60D, 90D, 180D
  • 1Y, 1.5Y, 2Y
  • Custom length (user-defined)
  • Displays all correlations in a compact table
  • Automatically highlights the highest correlation window

3. Plots the blended liquidity model directly on the main chart

  • Uses the right-side scale
  • Clean visual overlay
  • Helps compare liquidity shifts against price structure

4. Maintains transparency and reproducibility

  • All data sourced from public FRED datasets
  • No proprietary formulas
  • Designed for research, modification, and extension

Why This Script Is Useful

Traditional liquidity charts often plot only one variable at a time. This model adds clarity by:

  1. Combining multiple macro liquidity components into a unified metric
  2. Allowing control over which components are included
  3. Providing a smooth blended output that is easy to compare with price
  4. Offering multi-window correlation analysis to measure relationships
  5. Helping explore whether liquidity leads, lags, or coincides with price

This tool is ideal for macro-focused traders, swing traders, crypto/macro analysts, and anyone studying how systemic liquidity impacts market behavior.

Important Notes

  • FRED data updates weekly and may revise historical values
  • Correlation does not imply causation
  • Liquidity models are approximations, not definitive forecasts
  • Always validate settings on multiple timeframes and assets
