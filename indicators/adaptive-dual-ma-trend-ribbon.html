Adaptive Dual MA Trend Ribbon Community “Larsson-Style” Presets

Many traders enjoy experimenting with dual-moving-average “trend ribbon” structures that resemble the visual style of popular tools such as the Larsson Line. This indicator provides a flexible, well-documented version of that style for TradingView. The indicator is free to use; the script is protected.

How Traders Recreate This Without the Indicator

At its core, the different “Larsson-style” trend ribbons are based on a very simple idea: plotting two smoothed moving averages and observing how they interact over time.

If you want to recreate a similar structure manually in TradingView—without using this indicator—you can do the following:

  1. Add a Smoothed Moving Average (SMMA) to your chart.
  2. Set its length to 15 and leave the remaining inputs at their defaults.
  3. Add a second SMMA with the length set to 29.
  4. When the faster SMMA (15) moves above the slower SMMA (29), bias is bullish.
  5. When the faster SMMA (15) moves below the slower SMMA (29), bias is bearish.

Because moving averages lag, this tends to work best on daily and higher timeframes and during clear trends. It performs poorly in chop, so many traders combine it with structure/volatility filters for confirmation.

Overview

This script does not replicate, reverse-engineer, or replace any closed-source indicator. Instead, it provides a flexible framework for exploring trend-ribbon behavior using standard moving-average and ATR concepts.

What the Script Does

  • Plots fast/slow moving averages using SMMA (RMA), EMA, SMA, or WMA.
  • Colors the ribbon to show directional bias (bullish/bearish/neutral).
  • Optional ATR neutral zone filtering to reduce chop.
  • Optional bar coloring by trend state.

Why This Script Is Useful

Traditional moving averages identify trend direction but often lack context during uncertainty. This ribbon adds structure, neutrality filtering, and flexibility for experimenting with settings.

Important Notes

  • Free to use on TradingView; script is protected.
  • Moving averages lag and perform poorly in sideways markets.
  • Use alongside other technical tools for confluence.
